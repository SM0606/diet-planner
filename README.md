# Diet Planner App

A React web app for tracking meals, exercise, calories, protein, and generating a budget shopping list.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Upload this folder to a GitHub repository.
2. Go to Vercel and choose "Add New Project".
3. Import your GitHub repository.
4. Click Deploy.
5. Vercel will give you a public URL.
